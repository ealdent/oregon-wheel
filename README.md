# Fun with AI

A collection of games, tools, and experiments built with AI — hosted on GitHub Pages.

## Collections

- **[Arcade](arcade/)** — Interactive games & simulations (Genesis Colony, Life Lab, Pale Blue Dot, Baseboard Bandit)
- **[Tower Defense](tower-def/)** — AI-generated tower defense variants from multiple models
- **[Vector Defense](vector-defense/)** — Retro missile command with terrain mechanics
- **[SVG Forest](svg-forest/)** — Explorable SVG art worlds generated by different AI models
- **[Depths](depths/)** — Spinners, trackers & curiosities
- **[Utils](utils/)** — Infographics & visual tools
- **[Energy Model](energy-model/)** — US data center & grid capacity visualizer

## Deploy on GitHub Pages

1. Push to a public repo.
2. In **Settings → Pages**, set **Source** to "Deploy from a branch", pick `main` and `/ (root)`.
3. Site appears at `https://<user>.github.io/<repo>/`.

No build step needed — all static HTML.
